The Layer-by-Layer Method Explained

To prepare for solving the cube, here is some essential Rubik’s Cube knowledge to keep in mind.

Clockwise Turning Codes

For each turning code, it represent turning clockwise in the corresponding direction.

Counterclockwise Turning Codes

For each turning code followed by an apostrophe ('), it represents a counterclockwise rotation in the corresponding direction.
(i.e. R' turn is called as R prime.)

180-degree Turning Codes

For each turning code followed by a 2, it represents a 180-degree rotation in the corresponding direction.
